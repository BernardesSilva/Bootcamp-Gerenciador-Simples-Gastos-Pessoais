# 💰 FinanceX - Gestão Financeira Inteligente

O **FinanceX** é uma aplicação web de controle financeiro pessoal projetada para ajudar o usuário a monitorar gastos, definir metas de investimento e visualizar a saúde do seu orçamento de forma simples e intuitiva.

---

## 🚀 Proposta da Solução
Muitas pessoas perdem o controle de suas finanças por falta de uma ferramenta prática que permita o registro imediato. O FinanceX resolve isso com uma interface focada em agilidade, permitindo que o usuário:
* Registre gastos por categorias no momento em que acontecem.
* Defina aportes de investimento que são descontados automaticamente do saldo disponível.
* Visualize através de gráficos para onde o dinheiro está indo.

## 🎯 Público-Alvo
Indivíduos que buscam uma ferramenta leve e direta, sem a complexidade de cadastros bancários, ideal para quem quer organizar o orçamento mensal "na ponta do lápis" de forma digital.

## ✨ Funcionalidades Principais
* **Gestão de Renda:** Definição de orçamento mensal total.
* **Aportes Acumulativos (v1.1.0):** Sistema para somar novos investimentos a uma meta mensal, registrando o progresso.
* **Categorização Automática:** Divisão de despesas entre Alimentação, Transporte, Lazer e Outros.
* **Gráficos Dinâmicos:** Visualização em tempo real da distribuição de gastos via Chart.js.
* **Histórico de Meses:** Recurso de "Finalizar Mês" para arquivar dados e limpar a tela para o próximo ciclo.
* **Persistência Local:** Utilização de `localStorage` para manter os dados mesmo após fechar o navegador.

## 🛠 Tecnologias Utilizadas
* **HTML5 & CSS3:** Estrutura e estilização com interface Dark Mode.
* **JavaScript (ES6+):** Lógica de cálculos, manipulação de Array e DOM.
* **Chart.js:** Biblioteca para renderização dos gráficos de rosca.
* **Google Fonts:** Tipografia 'Poppins' para uma estética moderna.

## 📥 Instalação e Execução
Como o projeto é puramente Front-end, não é necessário instalar dependências de servidor ou bancos de dados externos.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/BernardesSilva/Bootcamp-Gerenciador-Simples-Gastos-Pessoais.git](https://github.com/BernardesSilva/Bootcamp-Gerenciador-Simples-Gastos-Pessoais.git)
    ```
2.  **Navegue até a pasta:**
    ```bash
    cd Bootcamp-Gerenciador-Simples-Gastos-Pessoais/src
    ```
3.  **Abra o projeto:**
    Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## 📖 Como Usar
1.  **Defina sua Renda:** Insira o valor total disponível para o mês.
2.  **Registre Gastos:** Preencha descrição, valor e categoria para ver o gráfico atualizar.
3.  **Gerencie Investimentos:** Defina uma meta e use o campo "+ Aporte" para adicionar reservas financeiras.
4.  **Encerre o Ciclo:** Ao final do mês, utilize o botão "Finalizar Mês" para salvar tudo no Histórico.
